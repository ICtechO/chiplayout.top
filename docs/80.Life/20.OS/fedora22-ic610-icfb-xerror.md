## 解决
按网上说的 
```sh
[admin@chiplayout ~]$ sudo gedit /etc/X11/xorg.conf.d/00-keyboard.conf
```
 加入下面的代码 
 ```sh
 Section "Extensions" Option "Composite" "Disable" EndSection
 ```
 
 ## 补充
 结果是直接进不了桌面了 后来发现，在.bashrc文件中加入下面这句可以完美解决。 
 ```sh
 export XLIB_SKIP_ARGB_VISUALS=1
 ```
