## 添加chrome源
```sh
su root cd /etc/yum.repos.d/
```
## 下载google-chrome.repo并保存
[google-chrome-mirrors.repo 百度网盘下载](http://pan.baidu.com/s/1eSsucnw)

## 安装Corome
```sh
dnf install -y google-chrome-stable
```
Terminal中显示完成时就可以按win键看见了
